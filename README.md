# Dev-Portfolio

## Overview

**Dev-Portfolio** is a modern, responsive developer portfolio built with React, TypeScript, and Tailwind CSS. It showcases your skills, projects, and experience, providing a clean and professional online presence.

## Live Demo

Check out the live portfolio here: [https://goodnessw.vercel.app](#)

## Features

- Personal **About Me** section  
- Showcase of **Projects** with links and descriptions  
- **Contact Form** for potential clients or collaborators  
- Fully **responsive design** for mobile, tablet, and desktop  
- Clean and modern UI powered by **shadcn-ui**  
- Easy to customize and extend  

## Technologies Used

- **React** – Component-based UI library  
- **TypeScript** – Type-safe JavaScript  
- **Tailwind CSS** – Utility-first styling  
- **Vite** – Fast development server and build tool  
- **shadcn-ui** – Prebuilt UI components  

## Getting Started

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd Dev-Portfolio

# Install dependencies
npm install

## Getting Started

# Start the development server
npm run dev
The portfolio should now be running at http://localhost:5173.

## Building for Production
# Build the project
npm run build

# Preview the production build
npm run preview

Customization

About Me: Update your personal bio and photo in the src/components/About folder

Projects: Add or edit project cards in the src/components/Projects folder

Contact Form: Customize form settings in src/components/Contact

Styling: Modify Tailwind CSS classes to change the design

Deployment

You can deploy your portfolio to any static hosting service such as Vercel, Netlify, or GitHub Pages.
